# AGENTS.md

## 项目名称
零基础AI项目教练

## 项目定位
这是一个面向零代码基础用户的 Web App，核心目标是：
1. 帮用户选择合适的 AI 编程路线（模型 + 编程工具）
2. 帮用户完成安装、接入和最小命令训练
3. 帮用户把模糊的软件想法变成项目设计书
4. 帮用户生成任务拆解和开发起步 prompt

## 默认技术栈
- Next.js（App Router）
- TypeScript
- Tailwind CSS
- shadcn/ui
- Prisma
- PostgreSQL

## 开发原则
- 一次只执行一个 task 文件
- 未经明确要求，不要跨 task 扩展功能
- 先做最小闭环，再补增强功能
- 中文界面文案优先
- 组件命名清晰，结构简洁
- 避免过度抽象和过度设计
- 静态数据优先放到独立配置文件
- 新增功能前先说明会修改哪些文件
- 完成后列出新增和修改的文件
- 不要擅自引入数据库、登录、支付、复杂状态管理，除非当前 task 明确要求

## UI 原则
- 面向普通用户，不要像开发者后台
- 简洁、清晰、留白充足
- 先保证可用，再做复杂视觉
- 优先保证首页、路线页、聊天页、设计书页的清晰度

## 代码原则
- 保持文件结构整洁
- 尽量写必要注释
- 不修改无关文件
- 不顺手重构与当前任务无关的代码
- 优先使用可维护的简单方案

## 任务执行流程
每次开始新任务时，请先：
1. 总结本次将创建或修改哪些页面、组件和文件
2. 说明不会实现哪些内容
3. 等待确认后再开始大规模改动

## 任务文件规则
- task.md 只作为任务索引和任务制度说明
- 每个具体开发任务单独放在 task1.md、task2.md、task3.md ...
- 执行时只读取当前被指定的那个 task 文件
